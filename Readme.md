## Node EV3 Tank

Control your Lego Mindstorms EV3 Tanks with an xbox controller over bluetooth

## Install

    git clone https://github.com/andrew/node-ev3-tank.git
    cd node-ev3-tank
    npm install
    node index.js

## Usage

`Left stick` for driving around, `A` for shoot gun.
